快速上手

只需在git clone命令中将克隆地址中 github.com 替换为 gitclone.com 即可

效果显著

首次clone需从 github 上缓存，以后克隆即可从缓存中直接拉取

方法一（直接clone）

Administrator@9QDZCH8M0W0EK2S MINGW64 ~/Desktop

$ git clone https://gitclone.com/github.com/little51/codechain

方法二（设置git参数）

Administrator@9QDZCH8M0W0EK2S MINGW64 ~/Desktop

$ git config --global url."https://gitclone.com/".insteadOf https://

$ git clone https://github.com/tendermint/tendermint.git
