# Gitダウンロードします

+ <a href="https://www.gitclone.com/download/Git-2.35.1.2-32-bit.exe">Git.exe(Windows_32bit)</a>
+ <a href="https://www.gitclone.com/download/Git-2.35.1.2-64-bit.exe">Git.exe(Windows_64bit)</a>
